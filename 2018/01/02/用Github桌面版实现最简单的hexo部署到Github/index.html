字数统计: 370 字 | 阅读时长 ≈ 1 分钟

首先，下载github桌面版

第二步，用github桌面版（要登陆）把项目clone下来

clone到一个空目录，之后就把hexo根目录下的文件全都复制黏贴到这个clone下来的目录下就好了，这样做不会有权限的冲突（哈哈哈哈）

再后一步，安装hexo-deployer-git

这一步很简单，在根目录执行如下命令

npm install hexo-deployer-git –save
